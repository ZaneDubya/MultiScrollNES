Each frame, the engine goes through all dynamics and statics and determineswhich, if any, should be drawn as sprites..alias	page_index.alias	page_copy.alias	page_bit.alias	bit_index	`SetByte page_index, $01	`SetByte bit_index, $0fcheck_page:	`SetByte page_bit, $80	ldx page_index	lda Dynamic_Page0,x	sta page_copycheck_bit:	lda page_bit	and page_copy	beq next_bit	; draw dynamic with x = bit_indexnext_bit:	dec bit_index	lsr	bcc check_bitnext_page:	dec page_index	bpl check_page
