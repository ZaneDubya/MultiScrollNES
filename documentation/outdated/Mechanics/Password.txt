32 Character alphabet: 5 bits of data per character.
	0123456789ACDEFGHJKMNPRSTVWXYZ?*
24 Character password: 120 bits of data.

Char Struct:
Bits	Description
2		Base class (4 possibilities)
2		Core item equipped
2		Defense item equipped
2		Attack item equipped
8		Level up choices (Level is taken from Progression)
----	
16		Total

Save data:
Bits	Description
8		Rnd Seed (for equipment choices and names)
8		Chksum
16		Currency
64		4 x Characters
24		Progression Flags
